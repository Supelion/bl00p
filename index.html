Bloop - A Hypixel Stats Bot

Open-Source!




Commands

bedwars - Returns the BedWars stats of a specified player.

skywars - Returns the SkyWars stats of a specified player.

duels - Returns the Duels Stats of a specified player.

profile - Returns the user's general Hypixel stats.

wdr - Returns the Watchdog ban statistic on Hypixel.

socials - Returns the socials linked to a player's minecraft account on Hypixel.

server - Returns the server status of a specified server IP.

skin - Returns the minecraft skin of a specified player.

compare - Compares two player's BedWars stats.

uuid - Returns the UUID of a specified player.

map - Returns a picture of the map specified.

requeue - Tells you if you should requeue or not.

key - Shows the statistics of the provided API key (the data is not stored!).

achievement - Generates a image of a minecraft achievement using parameters provided by the user.



src - Github Link for the bot's repository.

ping - Shows the bot's ping to the discord servers.

suggest - Takes the user's input for a suggestion for the bot.

invite - Shows the bot's invite link.

stats - Shows the bot's statistics.

help - Shows the bot's help command.
